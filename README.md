# Pearlsonal Project

## Description
Create an app for searching for food deals/cheap eats based on location. 

#### Repo
https://github.com/PearlR/pearlsonal-project

## Learning Objectives
- Databases
- Mobile App Set Up
- API's (Zomato, Google maps)
- Mobile first development

### Stretches

- Add user logins
- Add user database joins
- React
